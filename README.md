# Вертикальный раннер на Phaser + Vite

Этот проект — минимальный скелет вертикального раннера на Phaser 3 с рендерингом на примитивах (прямоугольники). В игре есть меню, игровой экран и экран окончания игры. Управление: пробел или тап/клик для прыжка.

## Версии

- Phaser 3.90.0
- Vite 6.3.1

## Что изменилось

- Игра переведена на вертикальный формат 720x1280 с масштабированием FIT и автоцентрированием.
- Добавлены сцены главного меню, игры и завершения с русскими подсказками.
- Реализован базовый раннер: игрок, пол, препятствия (земля и воздух), ускорение и счёт.

## Установка и запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите локальный сервер разработки:

```bash
npm run dev
```

После запуска откройте адрес, который покажет Vite (обычно `http://localhost:8080`).

## Управление

- Прыжок: **Space** или **тап/клик** по экрану.

## Структура проекта

- `src/main.js` — запуск приложения.
- `src/game/main.js` — конфигурация Phaser.
- `src/game/scenes` — сцены игры.
