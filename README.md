# Moon Runner (Phaser + Vite)

Этот проект — каркас вертикального раннера на Phaser 3. Игра запускается даже без ассетов: если картинок нет, используются примитивы (прямоугольники), поэтому сборка не ломается.

Для запуска установите зависимости (`npm install`) и запустите dev-сервер (`npm run dev`). Ассеты (если есть) кладите в `public/assets/moonrunner/` по структуре, указанной в `src/config/assetManifest.js`. При наличии файлов они автоматически подхватятся, при отсутствии — игра продолжит работать на плейсхолдерах.
